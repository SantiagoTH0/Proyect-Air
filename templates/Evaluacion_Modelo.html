Evaluación del Modelo

Métricas de desempeño

Dado que el objetivo del modelo es predecir la calidad del aire clasificándola en diferentes categorías, se seleccionaron las siguientes métricas de clasificación:

  • Accuracy: Proporción total de predicciones correctas.
  • Precision: Capacidad del modelo para no clasificar incorrectamente una clase.
  • Recall: Capacidad del modelo para detectar correctamente los casos positivos de cada clase.
  • Matriz de Confusión: Permite visualizar el rendimiento de clasificación por clase y detectar errores sistemáticos.

Estas métricas se generaron tras predecir con el conjunto de prueba (20%) no utilizado en el entrenamiento, asegurando una evaluación realista del rendimiento del modelo.

Validación cruzada

No se aplicó una validación cruzada tradicional en esta versión del proyecto. Sin embargo, la partición 80/20 garantiza una validación básica separando los datos en subconjuntos independientes. Se considera incluir validación cruzada en futuras iteraciones si se quiere obtener una estimación más robusta del rendimiento del modelo.

Análisis de Riesgos

Se identificaron y mitigaron posibles riesgos como:

  • Sobreajuste: Se limitó la profundidad del modelo (max_depth=10) y se estableció un número razonable de árboles (n_estimators=100) para evitar que el modelo memorice los datos.
  • Subajuste: Se verificó que el modelo tuviera un rendimiento aceptable en precisión y recall, lo cual indica que está capturando patrones significativos.
  • Sesgos: Se realizó limpieza previa de errores de codificación (por ejemplo, GIRÃ“N) y se verificó la consistencia de los datos para evitar sesgos por calidad de entrada.

Revisión con expertos

Actualmente, no se ha realizado una validación formal con expertos en calidad del aire. Sin embargo, el modelo está preparado para ser revisado por especialistas ambientales que puedan:

  • Validar la lógica de las predicciones.
  • Determinar si las variables para la predicción calidad del aire utilizadas son adecuadas y comprensibles.
  • Sugerir ajustes adicionales
